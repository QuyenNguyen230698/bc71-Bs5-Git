

Repository local (trong máy user)

Repository remote (trên github)

* tạo Repository local : git init (mỗi dự án làm 1 lần)

push : đưa source từ máy (local) len github (remote)

* liên kết repos local và repo remote : git remote add origin https://github.com/QuyenNguyen230698/bc71-Bs5-Git.git

* push gồm 3 bước (chọn, xác nhận, gửi)
     B1: git add -A
     B2: git commit -m " nội dung của commit"
     B3: git push -u origin master (lần đầu) / git push (lần 2 trở đi)
